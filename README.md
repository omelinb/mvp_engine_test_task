MVP Engine Test Task
-----------

System Dependencies
-------------------

- Ruby 2.6.5 (install with [rbenv](https://github.com/sstephenson/rbenv))

Project Install
-------------------
```shell
git clone git@github.com:omelinb/mvp_engine_test_task.git
```

Run tests
-----------------

```
ruby get_pairs_test.rb
```
